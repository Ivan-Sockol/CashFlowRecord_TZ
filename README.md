## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/Ivan-Sockol/CashFlowRecord_TZ.git
cd CashFlowRecord_TZ
```
### 2. Установка зависимостей

```bash
# Создание виртуального окружения
python -m venv venv

# Активация (Windows)
venv\Scripts\activate

# Активация (MacOS/Linux)
source venv/bin/activate

# Установите все необходимые зависимости:
pip install -r requirements.txt
```

### 3. Настройка базы данных

```bash
# Выполните миграции для создания таблиц в базе данных:
python manage.py migrate
```
### Примечание: По умолчанию используется SQLite. Файл базы данных db.sqlite3 создастся автоматически в корне проекта.
Если вы используете другую СУБД (например, PostgreSQL), перед выполнением миграций настройте подключение в файле settings.py
и создайте базу данных вручную.
### 4. Запуск веб-сервиса
```bash
python manage.py runserver
```

### После запуска сервис будет доступен по адресу:

```bash
http://127.0.0.1:8000/
```
## 🔑 Доступ к админ-панели
### Для входа в административную панель Django создайте суперпользователя:
```bash
python manage.py createsuperuser
```

### Следуйте инструкциям в терминале (введите имя пользователя, email и пароль).
### После создания суперпользователя перейдите по адресу:

```bash
http://127.0.0.1:8000/admin/
```

### Введите логин и пароль, чтобы получить доступ к админ-панели.
